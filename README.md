# Imui

一个基于极光Aurora IMUI即时通讯,扩展的（IM）UI 库
提供了消息列表、输入视图等常用组件，支持常见的消息类型：文字、图片、语音、视频，文件，地理位置等，并且修复源库中一些bug。
